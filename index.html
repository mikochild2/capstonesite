Task Monster

Age: 3+, Ads: None!


Make transitions easier!

A visual schedule helps children understand what to expect in their day.

  • Make the child’s daily routines more fun and interactive.
  • Reduce the stress of transitioning between activities.
  • Help the child understand what is happening now and what is coming next.

Features

Great for parents, teachers, and kids

  • Easily add tasks with images.
  • Start a count down timer to visually understand how much time is left in an activity.
  • Delete tasks when complete.

PECS System

We use Picture Exchange Communication System (PECS) from mypecs.com so the children will have fun pictures that easily communicate everyday activities.
